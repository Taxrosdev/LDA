# LDA

LDA is a simple yet powerful open source bot that you can use to connect a discord bot to Ollama

## Features

- No local database
- Simple setup

## Setup

### Docker

An example compose is available [here](https://github.com/Taxrosdev/LDA/blob/main/compose.yml)

### Development

```bash
git clone https://github.com/Taxrosdev/LDA.git
cd LDA
bun install
bun run .
```
